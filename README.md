# TodoApp backend

Backend server for the [todoapp react app](https://google.com) built using the MERN approach.

## APIs endpoints

| APIS             | ENDPOINTS  |
| ---------------- | ---------- |
| GET all tasks    | /tasks     |
| POST a task      | /tasks     |
| DELETE all tasks | /tasks     |
| PATCH a task     | /tasks/:id |
| DELETE a task    | /tasks/:id |
